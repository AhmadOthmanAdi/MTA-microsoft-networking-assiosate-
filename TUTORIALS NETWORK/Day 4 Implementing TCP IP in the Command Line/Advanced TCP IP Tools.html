Advanced TCP/IP Tools

  • NetStat

Display active TCP Connections & Ports where the Client is Listening / Ethernet Statistics / IP Routing Table

IPv4 Statistics for protocols ICMP;TCP;UDP

IPv6 Statistics for ICMPv6;TCP Over IPv6;UDP Over IPv6

WITH NO PAREMITERS (Display Active TCP Connections > connection to Other Systems)

  • NbtStat

Display NetBIOS over TCP/IP (NetBT) Protocols statistics for Local & Remote Clients

Display NetBIOS Name Tables for Local & Remote Clients

Display NetBIOS Cash

ESTABLISHES

Logical Names on the network

Sessions between Two Logical Names on Network > Establishing a session allows Two Clients on Network to start Reliable Data transfer

  • PathPING

Additional Information Displayed in Combination of Powers of PING and traceRT

Send's packages to Each Router On the way to Final Destination, and compute results based on RETURN time from each HOP

📍 PinPOINT Malfunctioned Routes/Routers/Lines By showing Packet Loss Degree

  • NsLookup

Diagnose DNS ISSUES

Lookup NAME of HOST to Check proper DNS Name Resolution

  • Netsh — Net Shell

Scripting Utility > With an extra commands e.g Netsh +Command allows to perform Different Options e.g <Local/Remote Client Commands> Provide Scripting Future < .BATCH > a Group of Scripts against a specific Target aka. Client on Network <Local or Remote>

Archiving Code Is possible with Export
  • Route (- Print)

Defines and Modifies Entries of local IP Routing Table e.g figure out way from city to other city

Display Routing Table with <<Route (-PRINT)> ~ NetStat - R>

Can be Used to Add/delete Static Routes

  • Net

Mini Command - Can be used to TEST Network Connectivity on System - not necessary a part of TCP/IP STACK

  • Telnet PORT:23

Allows To Communicate with remote Client also Using Telnet Protocol

Initiate TELNET Session CMD>Telnet> ..hostname"/..IP address" then a TELNET prompt is Received<low lever> to perform Basic Commands on Target Client

TEST Ports (Troubleshooting)> TelNET Server ..PORT NAME"
