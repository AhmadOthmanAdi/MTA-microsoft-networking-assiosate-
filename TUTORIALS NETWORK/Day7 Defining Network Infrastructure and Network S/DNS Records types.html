DNS Records types

(A records ) Store host name & its corresponding IPV4 Address / maps host names to addresses

(AAAA) Store Hostname and Corresponding IPV6 Address

(CNAME) Canonical name record can be used alias host name to another host name

(MX) Mail exchange specifies an SMTP simple mail transfer protocol for the DOMAIN, used to Route Outgoing Emails to Mail Server

(NS record) special DNS zone e.g Google.com is dedicated to a specific Name-Server and provide the address of the Name-Server

(PTR-record Pointer) allows DNS resolver to provide IP address and receive hostname Reverse-DNS-Lookup

(CERT-Record) Stores encryption certificates PKIX,SPKI,PGP

(SRV-Record) service location Similar to MX work on other communation protocols

(TXT-record) carries machine readable data such as encryption, sender policy framework, DKIM, DMARC

(SOA-record) Start of Authority: Appears on beginning of DNS zone file and shows the Authoritative Name-Server for the current DNS zone

specifies the host that is the authority for a given domain

(PTR-Records) used for Reverse DNS Using IP get Assosiated Domain / Hostname
